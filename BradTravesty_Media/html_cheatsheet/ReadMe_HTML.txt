Hello World
HTML file must end with .html extension
Runs in web browser (Chrome,Firefox)
 index.html is the root/home page of a website
 
Tag Syntax
<tagname>content</tagname>
Element names are surrounded by angle brackets
End tag is usually the same but with a forward slash
Some tags close themselves 

<h1>About US</h1>
<p>This is a paragraph</p>

<br/>(self closing XHTML style)
<br>(fine in HTML5)


HTML5 standard DOCTYPE Structure
<!DOCTYPE html>  
<html> 
<head>

<title>Page title</title>

</head>


<body>

<h1>My first heading</h1>
<p>My first paragraph</p>
</body>

</html>


TAG ATTRIBUES:
All tags can have attributes to provide info about an element
Placed within the start tag
key/value pairs (id ="someID")
Are always used as key value pairs
<tagname attributename="attributevalue"></tagname>

<h1 title ="My Company">About Us</h1>


TABLES:
